# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities to security@klever.io.

Do not create public issues for security vulnerabilities. We will acknowledge your report within a reasonable timeframe and work to resolve the issue promptly.

## Disclosure Policy

We follow responsible disclosure practices. We will coordinate a fix prior to public disclosure whenever possible.

## Additional Notes

- If the vulnerability affects other Klever projects, we may coordinate across repositories.
- Please include as much detail as possible (affected versions, reproduction steps, potential impact).
