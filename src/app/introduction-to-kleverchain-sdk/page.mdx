# Introduction to Klever Blockchain SDK
Introduction to develop and start integration with Klever Blockchain main methods

The SDK module provides a quick and easy way to make contract calls directly to the Klever blockchain, either using the Klever wallet or a `.pem` file in a node server.

# Installation

## Web App

```bash
$ npm install @klever/sdk-web
```

or

```bash
$ yarn add @klever/sdk-web
```

you can also import the script directly from [this url](https://kleverchain-sdk-web.s3.us-east-1.amazonaws.com/kleverchain-sdk-web-esm-1-0-x.js), as [exemplified here](/web-app#vanilla-js).

## NodeJS

NodeJS must be above version 18.0.0

```bash
$ npm install @klever/sdk-node
```

or

```bash
$ yarn add @klever/sdk-node
```
